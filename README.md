swiftsample
===========

SwiftでSpriteKitを使って簡単なサンプルを作ってみました

* タップするとランダムな色とサイズでボールを置きます
* ボールをタップすると消えます
* 傾けた方にころがっていきます
* シェイクするとボールが全て消えます
